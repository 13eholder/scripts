# Available Scripts

- `ls.sh`: List All Avalible Scripts [See More](ls.sh)
- `net/clash_port.sh`: This script checks for processes that are listening on ports associated with "clash". [See More](net/check_clash_port.sh)
- `net/default_proxy.sh`: This script checks for processes listening on common proxy ports (9090, 7892, 7891, 7890). [See More](net/check_default_proxy.sh)
- `app/reinstall_gemini.sh`: This script uninstalls and then reinstalls the `@google/gemini-cli` globally to ensure the latest version is used. [See More](app/reinstall_gemini.sh)
- `hello.sh`: A simple script that prints "Hello, World!" to the console. [See More](hello.sh)
